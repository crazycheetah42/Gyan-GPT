Download GyanGPT for Linux

Version: 2.1.0


To download GyanGPT for Linux, you will need to have at least Python 3.8 installed. You will also need to have pip installed.
You can install pip by running this command (for Debian/Ubuntu-based distributions): sudo apt install python3-pip -y
Next, you will have to run these three commands (all distributions):
git clone https://github.com/crazycheetah42/GyanGPT
cd GyanGPT
sudo apt install portaudio19-dev python3-tk -y
sudo pip3 install tk PyAudio SpeechRecognition openai
python3 GyanGPT_linux.py
And then you will be able to use GyanGPT.
